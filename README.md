# SmartSupplyAI.net RSS Feed

SmartSupplyAI.net podcast RSS feed for automated distribution to Spotify.

## Files

- `smartSupplyAI.rss` - RSS feed for the podcast
- `smartSupplyAI-podcast-Week-13_Complete.mp3` - Episode audio file

## RSS Feed URL

https://raw.githubusercontent.com/fdynotifications-bot/smartSupplyAI-spotifeed/main/smartSupplyAI.rss

## Podcast

- **Title:** SmartSupplyAI.net
- **Host:** Fabrice Dutailly
- **Category:** Technology
- **Description:** Discover weekly the latest innovations in artificial intelligence applied to supply chain and logistics. Emerging trends and strategic insights to stay at the forefront of innovation.

## Automation

This RSS feed is used for automated distribution to Spotify via Spotify for Creators Distribution settings.

## Usage

For automated episode publishing:
1. Upload new MP3 file to this repository
2. Update `smartSupplyAI.rss` with new episode details
3. Commit and push to GitHub
4. Spotify automatically publishes the new episode from the RSS feed
